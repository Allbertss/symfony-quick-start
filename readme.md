# Symfony quick start

## Repository contains

- Twig bundle
- Profiler
- Logger
- Apache-pack
- Webpack Encore
  - jQuery
  - Stimulus js
  - bootstrap

### After cloning project

```shell
composer install
```

```shell
yarn install
```

```shell
yarn watch
```

```shell
symfony check:req
```

*Solve problems if any*

### Versions

- Symfony 6.1.6
- Node 16.18
- npm 8.19.2 *or* Yarn 1.22.19
